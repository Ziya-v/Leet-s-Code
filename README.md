## 🧠 My Problem-Solving Process

For every problem, I try to follow a consistent approach:

1. Understand the problem statement and constraints.
2. Identify patterns and suitable data structures.
3. Build a brute-force solution first.

6. Write clean, readable, and maintainable code.
